My Favorite Realizations

BookAround

It took three weeks to architect, design, code, and test this prototype, while in Brussels, Belgium. It was a pleasure to engineer this project, from start to finish, with the aid of three other talented individuals.

It was created using HTML5, CSS3, Bootstrap, JavaScript, PostgreSQL, Elastic Search, git, GitHub, and Heroku.

BookAround is currently being upgraded, and has a new look, but the same great freatures.

GVG Portfolio Site

This website was created using HTML5, CSS3, BootStrap, JavaScript, and Middleman, which uses embedded Ruby to help keep the codebase DRY. Git and GitHub were used to manage version control.

It was an enjoyable experience creating this web application and I hope to work on other similar projects.
